align

src/xy/align.js

Aligns data of two spectra by verifying wether x values are in a certain range (delta). The two spectra should not have two consecutive x values which difference is smaller than delta to achieve good results!

align(spectrum1: object<array>, spectrum2: object<array>, options: object)
Parameters
spectrum1 (object<array>) First spectrum data
spectrum2 (object<array>) Second spectrum data
options (object = {})
Name Description
options.delta number (default 1) The range in which the two x values of the spectra must be to be placed on the same line
options.common boolean (default true) If true , only the data considered as common to both spectra is kept. If false , the data y arrays are completed with zeroes where no common values are found
options.x string (default 'x1') Defines what x values should be kept ( x1 : spectrum 1 x values, x2 spectrum 2 x values, weighted : weighted average of both spectra x values)
options.weightFunction function (default undefined) Function that allows to weight delta depending on the X values of the spectrum
